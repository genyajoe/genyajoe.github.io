Distributed network flow algorithm balancing link loads

    This project proposes a distributed algorithm for congestion control in Software Defined Networking with distributed controllers by exploiting ring structures in a network and graph coloring. Our algorithm dissects a network into small ring units called tie-sets and updates flows on edges by enclosed calculation in each controller, which is in charge of one of the ring units. Additionally, it is shown that the deployment of coloring in a tie-set meta-graph enables the distributed algorithm to avoid update conflicts in shared edges with less communication among controllers.

Reliable tree composition

    This project aims at propounding a composition problem of a resilient communication tree on intermittent communication networks such as Disruption Tolerant Networks (DTN). Two types of edge weights called availability and commonality are introduced in order to evaluate the resiliency of trees. Availability represents a probabilistic stability of a communication link corresponding to an edge, and commonality indicates the utilization of each edge based on relative locations on a tree. It is shown that the trees satisfying the proposed ordering property relating to the two edge weights provide more stable connections among communication nodes.

Controller placement problem in Software Defined Networking (SDN)

    This project tries to propose an algorithm to determine a controller location on a given network considering loads of all communication nodes. The load on each node is defined by extending stress centrality, which indicates a load of each node based on the number of edge disjoint paths. Our results indicate that the placement alleviating the load on nodes also improves latency, survivability and path availability metrics.
